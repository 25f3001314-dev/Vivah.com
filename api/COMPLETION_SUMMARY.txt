## 🤝 Integration with Frontend

Your existing `/webapp/app.py` can connect to this API:

```python
# In webapp/app.py
import requests

BACKEND_API = "http://localhost:8000/api/v1"

@app.route('/api/milan', methods=['POST'])
def calculate_milan():
    boy_name = request.json.get('boy_name')
    girl_name = request.json.get('girl_name')
    
    # Call backend API
    response = requests.post(
        f"{BACKEND_API}/matchmaking/ashtakoot",
        json={"boy_name": boy_name, "girl_name": girl_name}
    )
    
    return response.json()
```

## 🐛 Troubleshooting

### Server won't start?
```bash
# Check if port 8000 is in use
lsof -i :8000

# Kill existing process
kill -9 <PID>

# Try again
python main.py
```

### "ModuleNotFoundError"?
```bash
# Make sure you're in the right directory
cd /workspaces/Vivah.com/api

# Try again
python main.py
```

### Names not working?
```bash
# Names must be in Devanagari script
✓ राहुल (correct)
✗ Rahul (incorrect - use Hindi/Devanagari)
```

### Port already in use?
```bash
# Find what's on port 8000
netstat -tlnp | grep 8000

# Kill the process
kill <PID>
```
